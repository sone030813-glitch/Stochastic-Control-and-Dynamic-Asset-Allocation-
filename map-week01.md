Key Derivations (The "Two Must-Knows")Case A: Log-Price ($\ln S$)Function: $f(S) = \ln S$ (Concave shape $\cap$).Second Derivative: $f_{xx} = -1/S^2$ (Negative).Result: Drift decreases.Formula: $d(\ln S) = (\mu - \frac{1}{2}\sigma^2)dt + \sigma dW$.Meaning: This transforms Geometric Brownian Motion into a Normal Distribution.Case B: Inverse Price ($1/S$)Function: $f(S) = 1/S$ (Convex shape $\cup$).Second Derivative: $f_{xx} = 2/S^3$ (Positive).Result: Drift increases (Jensen's Inequality).Formula: $d(1/S) = \frac{1}{S}(\sigma^2 - \mu)dt - \frac{\sigma}{S}dW$.Core Difference: The sign of $f_{xx}$ determines if we add or subtract drift.3. The Product Rule (Integration by Parts)The Skeleton Formula (Always Fixed)$d(XY) = X dY + Y dX + dX dY$The "Cross Term" ($dX dY$) CalculationThis term is not fixed; it depends on the source of randomness.Scenario 1 (Same Noise): If both have $dW$, then $dX dY = \sigma_1 \sigma_2 dt$.Scenario 2 (Independent Noise): If $dW_1 \perp dW_2$, then $dX dY = 0$.Scenario 3 (Deterministic): If one process has no $dW$ (e.g., Time $t$ or Bond $B$), then $dX dY = 0$.4. Financial Concepts & ApplicationIsolate the DriftTechnique: Group all terms with $dt$ into one bracket.Goal: To check if the process is a Martingale.Rule: If Drift (coefficient of $dt$) is 0, the process is a Martingale (Fair Game).Origin of Parameters$\mu$ comes from the Stock definition: $dS = \mu S dt + \sigma S dW$.$r$ comes from the Bond definition: $dB = r B dt$.Example: Discounted Stock Price ($S/B$)Using Product Rule on $S \cdot B^{-1}$.Drift changes from $\mu$ to $(\mu - r)$.Under Risk-Neutral Measure ($Q$), we force $\mu = r$ so the drift becomes 0.5.
